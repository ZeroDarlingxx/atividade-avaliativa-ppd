#  Jantar dos Filósofos

Implementações paralelas do clássico problema de sincronização **Jantar dos Filósofos**, desenvolvidas para a disciplina de Programação Paralela e Distribuída.

## Sobre o Problema

O "Jantar dos Filósofos" é um problema clássico de ciência da computação que ilustra desafios de sincronização e prevenção de deadlock em sistemas concorrentes. Cinco filósofos alternam entre pensar e comer, compartilhando cinco garfos dispostos entre eles.

##  Estrutura do Projeto
src/
 ├── tarefa1/
 ├── tarefa2/
 ├── tarefa3/
 ├── tarefa4/
 └── tarefa5/

Cada pasta contém uma solução independente para o problema.

Requisitos
Java JDK 8 ou superior

Compilação
A partir da pasta src:

javac tarefaX/*.java

(Substitua tarefaX pela tarefa desejada.)

Execução
java tarefaX.JantarDosFilosofos
Testes

Cada tarefa imprime logs detalhados da execução e estatísticas finais após o período configurado de execução.
